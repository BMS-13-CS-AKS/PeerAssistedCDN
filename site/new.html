Example of "thor" implementation:

  1. open console and execute the below commands
  2. var a = new thor([address of signaling server]);
  3. a.name = "whateveryoulike"
  4. a.start();
  5. do the above steps for another tab
  6. a.connect([other peers name]);
Thor was made for multiple connections so we can create multiple channels with ease.

Note: You can skip step 2 and thor object will choose its own random 20 character ids.
